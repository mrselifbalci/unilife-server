PAGINATION

The API will automatically paginate responses, limiting the number of documents returned per page to 10 by default. You can adjust the page size by including the "limit" parameter in the URL. For instance, to retrieve 5 documents per page, append "?limit=5" to the URL.

To access different pages, include the "page" parameter in the URL. If you don't specify a page number, the API will return the first page. For example, to retrieve the second page, append "?page=2" to the URL.

If you wish to include both the limit and page parameters, separate them with an ampersand "&" in the URL.

EXAMPLE:

https://unilife-server.herokuapp.com/cities?limit=5 https://unilife-server.herokuapp.com/cities?limit=5&page=2

Filter Properties with Query

METHOD: POST

DESCRIPTION:

The filter endpoint allows you to retrieve properties based on certain filters. To use this endpoint, send a POST request to the API with the required city_id parameter in the request body. Additionally, you can include any of the optional parameters: bedroom_count, bathroom_count, rent, and property_type.

When you use the filter endpoint, you can specify the minimum number of bedrooms and bathrooms you want the properties to have by including the bedroom_count and bathroom_count parameters in your request. The API will then return only those properties that meet or exceed the specified minimum number of bedrooms and bathrooms.

Additionally, you can specify a maximum rent amount by including the rent parameter in your request. The API will then return only those properties whose rent is less than or equal to the specified maximum rent amount.

For example, if you set bedroom_count=2, bathroom_count=1, and rent=1500 in your request, the API will only return properties that have at least 2 bedrooms, at least 1 bathroom, and a rent of no more than $1,500 per month.

It's important to note that if you don't include any of these parameters in your request, the API will return all properties in the specified city, regardless of the number of bedrooms and bathrooms or the rent amount.

REQUEST BODY:


                            
    query={
        city_id:"633a96b96893d471a68cc891",
        bedroom_count:4,
        bathroom_count:2,
        property_type:"Detached",
        rent:1500 
    }

ENDPOINT:

https://unilife-server.herokuapp.com/properties/filter
