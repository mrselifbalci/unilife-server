Pagination

The API will automatically paginate responses, limiting the number of documents returned per page to 10 by default. You can adjust the page size by including the "limit" parameter in the URL. For instance, to retrieve 5 documents per page, append "?limit=5" to the URL.

To access different pages, include the "page" parameter in the URL. If you don't specify a page number, the API will return the first page. For example, to retrieve the second page, append "?page=2" to the URL.

If you wish to include both the limit and page parameters, separate them with an ampersand "&" in the URL.
https://unilife-server.herokuapp.com/cities?limit=5 https://unilife-server.herokuapp.com/cities?limit=5&page=2
